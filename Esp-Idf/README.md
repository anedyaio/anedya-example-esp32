# Anedya-Example-ESP32 (IDF)

<p align="center">
    <img src="https://cdn.anedya.io/anedya_black_banner.png" alt="Logo">
</p>

Find examples helpful for using the Anedya SDK with ESP32 devices running ESP-IDF.

### Wi-Fi Connectivity Example
Learn how to connect your ESP32 to Anedya using built-in Wi-Fi. This example covers device initialization, Wi-Fi configuration, and data transmission over a standard internet connection.

[View Internal Wi-Fi Example](https://github.com/anedyaio/anedya-dev-sdk-esp-idf/tree/main/examples/example-application-internal-wifi)

### Cellular Connectivity Example
Explore how to establish a connection using the Quectel EC200u/g cellular module.

[View Quectel EC200u/g Example](https://github.com/anedyaio/anedya-dev-sdk-esp-idf/tree/main/examples/example-application-quectel-ec200)
